About Breast Cancer Prediction

This breast cancer prediction tool uses a machine learning model trained on the Wisconsin Breast Cancer Dataset. The model analyzes cell nuclei characteristics from a digitized image of a fine needle aspirate (FNA) of a breast mass to classify tumors as malignant or benign.

Input Parameters:
  • Mean values: Mean of the features computed for each cell nucleus
  • Error values: Standard error of the features computed for each cell nucleus
  • Worst values: "Worst" or largest mean value for each feature computed for each cell nucleus
